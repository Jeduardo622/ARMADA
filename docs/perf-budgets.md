# Performance Budgets (MVP Targets)

- FPS: 30fps target on mid-tier devices; p5 > 24fps.
- Memory: < 800MB runtime on mid-tier; monitor spikes.
- Load: cold start < 12s; mission load < 6s.
- Draw calls: target < 1.5k mid-fight; LODs required.
- Texture/asset: sail/ship skins atlased; texture sizes capped per platform; compressed formats mandatory.
- Network: payloads < 200KB per mission start/end; retry/backoff.
- QA: weekly perf runs on device matrix; regressions blocked at gates.
