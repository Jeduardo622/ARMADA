# Mission Review Checklist (MVP)

- Objectives clear (primary + 3-star goals); fail reasons explicit.
- Difficulty banded appropriately for chapter; tuning reviewed against balance tables.
- AI profiles assigned from `ai-profiles.md` with param overrides noted.
- Wind/terrain modifiers documented; no surprise difficulty spikes.
- Rewards aligned to progression pacing; no economy exploits.
- Telemetry fields set (mission_id, fail_reason taxonomy, damage profile expectations).
- Performance sanity on target devices; VFX budgets respected.
- Localization stubs added to central strings file.
- QA notes include edge cases and repro steps for scripted events.
