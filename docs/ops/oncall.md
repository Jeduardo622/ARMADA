# On-Call & Escalation

- Rotation: weekly primary + backup; schedule published 4 weeks ahead.
- Coverage: crash/perf regressions, flag/config incidents, payment/store issues.
- Escalation: primary → backup → PM/Product for player impact; Security for abuse/cheat vectors.
- Contact: ops channel + pager; acknowledge within 15 minutes during coverage hours.
- Handover: daily note of active incidents/flags; unresolved items tracked in RAID log.
